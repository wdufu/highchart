# highchart
功能强大、开源、美观、图表丰富、兼容绝大多数浏览器的纯js图表库
官方地址：http://www.hcharts.cn/

本项目主要是把highchart整合到具体项目开发中，让你很快上手，各种图标，各种样式在不同情况下的具体使用方法，复制一下就OK
